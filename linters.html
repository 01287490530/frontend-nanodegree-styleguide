Udacity Frontend Nanodegree Style Guide

What Are Linters?

According to WikiPedia, linters:

...are often used to find code that doesn't correspond to certain style guidelines. They can also be used as simple debuggers for common errors, or hard to find errors...

So a linter checks your code for accuracy of both syntax and style requirements.

Check out the following sections for information linting each file type:

  • Linting HTML
  • Linting CSS
  • Linting JavaScript
  • Running A Linter

Linting HTML

The linter Udacity uses is HTMLHint. The .htmlhintrc file contains the configuration object used to validate your HTML files.

The validation rules can be found on the Rules page of the project's repo.

Linting CSS

The linter Udacity uses is Stylelint. The .stylelintrc file contains the configuration object used to validate your CSS files.

All of the rules that Stylelint supports are located here: https://stylelint.io/user-guide/rules/. If you have a question about how a specific rule works, just add the rule to the end of that URL. So to view how length-zero-no-unit works, check out the link: https://stylelint.io/user-guide/rules/length-zero-no-unit.

Linting JavaScript

The linter Udacity uses is ESLint. The .eslintrc.json file contains the configuration object used to validate your JavaScript files.

All of the rules that ESLint supports are located here: http://eslint.org/docs/rules/. If you have a question about how a specific rule works, just add the rule to the end of that URL. So to view how no-trailing-spaces works, check out the link: http://eslint.org/docs/rules/no-trailing-spaces.

Running A Linter

The best way to develop with a linter is to have it active in your code editor while you're developing. To get the linters working in your editor:

  • make sure you have the linter file(s) in the root of your project's directory structure
  • install the necessary plugin/extension for your editor

Linter Plugins / Extensions

  • Atom
    • Linter plugin (required to run the following linter plugins)
    • linter-htmlhint plugin
    • linter-stylelint plugin
    • linter-eslint plugin
  • Sublime Text 3
    • SublimeLinter plugin (required to run the following linter plugins)
    • SublimeLinter-contrib-htmlhint plugin
    • SublimeLinter-contrib-stylelint plugin
    • SublimeLinter-contrib-eslint plugin
  • VS Code
    • HTMLHint extension
    • stylelint extension
    • ESLint extension
